Mini-program to calculate neutron energy spectrum from gold foil activation measurement
Used 15 foils in polyethylene block
Input needed:
- measured activation (add to code)
- input file with response functions (from simulations) in the same format as used by the unfolding programs MAXED and GRAVEL,
    (https://www.oecd-nea.org/tools/abstract/detail/nea-1665)
